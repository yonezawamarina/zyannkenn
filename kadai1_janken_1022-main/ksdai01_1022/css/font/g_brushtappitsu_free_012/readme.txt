■解説
年賀状、コミックのセリフや擬音等、和風イラストの文字入れ等に使用する事を想定した
筆文字風のttfフォントです。
フォント名は「たっぴつかっこわらい」と読みます。

ver.0.12での収録文字は
半角英数字・半角カナ
全角ひらがな・カタカナ・英数字およびShift-JIS領域の記号全部
漢字1059字（教育漢字1006字全部＋年賀用漢字＋その他の漢字若干）
となります。

ver.0.10→ver.0.11では教育漢字を追加しました。

ver.0.11→ver.0.12では「龍」の字を修正、および

「〜」（Unicode：301C 波ダッシュ）を追加しました。
ただし、この字は制作ソフトの仕様上、縦書きに対応していません。
不具合がある場合は「～」(Unicode：FF5E 全角チルダ)をお使い下さい。まったく同一の字形にしています。


　収録文字についてなどの詳細はこちらを参照ください。
　　→http://material.animehack.jp/font_brushtappitsu.html
